# Inherit – Indian Cultural Heritage Learning Platform

🔗 Live Demo: https://inherit-project.vercel.app/

## Overview
Inherit is a web application that promotes awareness of Indian culture, heritage, historical places, and famous monuments. The platform provides educational content, virtual tours, and interactive features to help users explore India's rich cultural history.

## Features
- Information about historical places and monuments
- Interactive learning content
- Virtual tour experience
- Cultural heritage resources

## User Roles

### Admin
- Manage platform content
- Maintain accurate information
- Monitor user engagement

### Cultural Enthusiast
- Explore heritage content
- Participate in virtual tours
- Engage in cultural discussions

### Content Creator
- Create and update educational content
- Develop interactive features
- Manage virtual tours

### Tour Guide
- Provide insights during virtual tours
- Answer user queries about cultural sites
